Proyecto de Software 2017 - Parcial 1era Fecha - 04/11/2017

Tema 2

Se requiere desarrollar un sistema para el recuento de votos de las elecciones Legislativas 2017 de la Provincia de Buenos Aires.  El recuento se realiza a partir de los telegramas confeccionados por los/las presidentes/as de mesa, quienes se encargan de llenar el formulario en soporte papel para una provincia determinada, en este caso Buenos Aires, un ejemplo podría ser:

Cada provincia está dividida en secciones (localidades), en la cual cada una se divide en circuitos y a su vez, cada circuito en mesas.  Para cada terna (Sección, Circuito y Mesa) se registra la cantidad de votos que recibió cada agrupación política, en cada una de las categorías (Senadores Nacionales, Diputados Nacionales, Diputados Provinciales y Concejales).

Se debe implementar la siguiente funcionalidad:

  • Login de usuario, se deberá ingresar nombre de usuario y contraseña, validando en el cliente y en el servidor, que los datos ingresados no estén blanco.
  • Una vez logueado el usuario, se debe poder listar el resultado de los votos realizados a las agrupaciones políticas, para esto previamente se deberá seleccionar la sección, circuito y mesa.

Aclaraciones:

  • Debe respetar MVC y debe controlar los datos enviados desde el cliente al servidor.
  • Se debe implementar con PDO y respetar las sesiones planteadas.
  • La vista debe ser implementada mediante Twig. 
  • Los datos ingresados por el visitante no deben ser susceptibles a SQL Injection ni XSS. Indicar en el código (con comentarios) dónde y cómo se evitan estas vulnerabilidades.
  • Realice las aclaraciones que crea necesarias.
